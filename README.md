# 🌌 JyotirVaani — Vedic Astrology & Personal Guidance  

![JyotirVaani Banner](assets/social-preview.jpg)  

> A modern, responsive astrology web app providing **personalized Vedic astrology readings, kundli generation, compatibility analysis, and practical life guidance**.  

---

## ✨ Features
- 📜 **Kundli / Birth Chart** generation  
- 💍 **Marriage Compatibility (Match Making)**  
- 💼 **Career & Life Guidance**  
- 📱 Responsive design (mobile-friendly)  
- 🔒 Simple & secure **booking form**  
- 🌐 SEO optimized with **OpenGraph + JSON-LD**  
- 🎨 Dark-theme UI with modern card-based layout
